<!-- Цель создания проекта, для какого курса, предмета выполнялся проект -->
### Задание в рамках проекта-идеи для стартапа _УМНИК_
<!--  Описание проекта, чему научилась, что попробовала реализовать в рамках создания проекта -->
> Стояла задача реализовать **лендинг** для демонстрации товара - умного ошейника для питомцев, используя макет- [ссылка](https://www.figma.com/proto/FuZm2YkPS3aTVktxzd9cx2/%D0%BB%D0%B5%D0%BD%D0%B4%D0%B8%D0%BD%D0%B3?node-id=0-1&t=2VXE7WBzTj3X1moV-1), созданный ранее, важные моменты в реализации:
> 1. Визуальная составляющая должна поддерживать добрую и позитивную историю отношений с питомцами
> 2. Ссылки визуально должны отличаться, считываться, как кликабельные.
> 3. Присутствует кликабельная навигация по сайту, все элементы макета реализованы в верных тегах
> 4. Стили легко масштабируются на другие разделы сайта
